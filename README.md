# Spring Boot REST API

This project is a backend REST API developed using Spring Boot.

## Features
- RESTful APIs using layered architecture
- JPA and Hibernate for database persistence
- Entity relationships and validations
- Global exception handling
- Logging for debugging and monitoring

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- MySQL (configurable)

## Notes
Spring Security concepts have been studied and understood but are not implemented in this version of the project.
